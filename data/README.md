# Data Directory

This directory contains SQLite database files and other data generated by the player tracker.

## Files

- `player_tracker.db` - Main database storing player sessions and statistics
- `player_tracker_*.db` - Backup database files (timestamped)

## Database Schema

The database contains the following tables:

- **train_sessions** - Train driving sessions with distance and points
- **station_sessions** - Dispatcher sessions at stations  
- **train_station_passages** - Stations passed during train sessions
- **steam_stats** - Historical Steam statistics snapshots

Use `scripts/show_summary.py` to view the tracked data.
